# 勤怠管理システム - クイックスタートガイド

## 🚀 すぐに始める

### 1. アプリケーション起動

```bash
npm run dev
```

### 2. ブラウザでアクセス

```
http://localhost:3000
```

### 3. ログイン

- **URL**: 自動的に `/login` にリダイレクト
- **パスワード**: `admin123`
- **ボタン**: 「ログイン」をクリック

### 4. メインページで勤怠管理

- 出勤・退勤の打刻
- 休憩時間の記録
- 今日の記録確認

### 5. ログアウト

- 右上の「ログアウト」ボタンをクリック

## 🔧 設定変更

### パスワード変更

`.env.local` ファイルを作成：

```bash
ADMIN_PASSWORD=your_new_password
```

## 📁 主要ファイル

- **ログインページ**: `/login`
- **メインページ**: `/`
- **管理画面**: `/admin`

## ⚠️ 注意事項

- 本番環境では強力なパスワードを設定してください
- セッションは24時間で期限切れになります
- 詳細な設定は `docs/AUTH_SYSTEM.md` を参照してください

---

**問題が発生した場合**: ブラウザのキャッシュとクッキーをクリアして再試行してください。
